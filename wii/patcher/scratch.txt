how do we handle the installation?
we want to extract the files from an ISO to the SD card
each mod will need its own copy unless we set up the game to load overrides from another directory
we should do that to save space

we want to be able to install multiple mods
eg Amethyst + some custom character
so we'd want to have a base directory with the unmodified game files, and then one override directory for each installation
so you can have multiple installations that each have different mod sets
